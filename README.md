 # IT Support Toolkit

## 📌 Description
A collection of Python scripts designed to assist with system diagnostics, troubleshooting, and automation of common IT support tasks.

## 🚀 Features
- System information checker  
- CPU and memory usage monitor  
- Network connectivity tester  
- Disk cleanup automation (coming soon)  

## 🛠️ Technologies Used
- Python  

## 📂 Project Structure
- scripts/ → contains all automation scripts  
- docs/ → documentation and guides  

## 🎯 Purpose
This project demonstrates practical IT support skills through automation and scripting.

- Expand diagnostic tools
